## üõ† Tech Stack (High-level Overview)

### Frontend
- React 18 + TypeScript  
- Vite  
- Capacitor 7 (iOS & Android builds)  
- Tailwind CSS  
- Radix UI  
- TanStack Query  
- React Hook Form + Zod

### Backend
- Vercel Serverless Functions (Node + TypeScript)
